Programs to derive contiguity relations for hypergeometric system of Horn type. 
See Hiromasa Nakayama, Nobuki Takayama, Comprehensive Restriction Algorithm for Hypergeometric Systems, 
( https://arxiv.org/abs/2510.05104 )
on algorithms implemented.

These programs will be included in asir-contrib (https://www.openxm.org) in a near future.
